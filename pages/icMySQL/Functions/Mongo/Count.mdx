# Count
The count function is to get the count of data in the specified table of the database.

## Params
- Database ID (OPTIONAL) is to specify for what database is the query to. By default it take DB#1 you can change the default DB in the config.js.
- Params(*) the query to execute.
- Function(*) is the function that will be executed when the query is finished.

## Return
The mongo functions never return any value, always execute the provided callback

## Use
- If you want to make the query to execute in the main thread you have to specify a simple function like this: `() => {}` to do anything depending on the language.
<Tabs items={["Lua", "JS", "C#"]}>
<Tab>
```lua
MySQL.Mongo.MongoCount(1, {
    collection = "users",
    query = {
        ["name"] = "kDex"
    },
}, function(result)
    print(json.encode(result));
end);
```
</Tab>
<Tab>
```js
MySQL.Mongo.MongoCount(1, {
    collection: "users",
    query: {
        name: "kDex"
    }
}, function(result) {
    console.log(JSON.stringify(result));
});
```
</Tab>
